# GTTracker
This project is design to track all the registered members mobile on Google maps using there android smartphone. The Google maps API are used to show all the devices on that are registered in web application which is running on your home. The smartphone contains android app which sends data at user define period to the web application. This tracking system especially design for parents to monitor their childrenâ€™s on the move. Consider a situation where a child lost a way to home or he or she meet with an accident in that case child use there tracker app to send alert notification to parents and the web application display it. The web application is design to handle various types of alert that broadcast from android app.
